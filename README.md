# React + TypeScript + Vite

## Установка

1. Установите зависимости:

  Если у вас установлен `npm`:

    ```bash
    npm install


  Или, если вы используете `yarn`:

    ```bash
    yarn install

## Запуск проекта

1. Запустите проект в режиме разработки:

  Если вы используете `npm`:

    ```bash
    npm run dev


  Или если вы используете `yarn`:

    ```bash
    yarn dev
